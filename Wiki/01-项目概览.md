# 项目概览

## 简介

**AutoCodeForge** 是一个 AI 驱动的研发流水线系统，采用前后端分离架构，旨在通过 AI Agent 编排和自动化任务执行提升软件开发效率。

## 核心功能

### 🤖 AI Agent 管理
- **Agent 注册与配置**: 支持多种 Agent 类型注册和技能配置
- **Agent 编排**: 基于任务需求自动选择和调度 Agent
- **Agent 通信**: Agent 间协作通信机制

### 📋 任务中心
- **任务 CRUD**: 创建、查询、更新、删除任务
- **任务步骤管理**: 分步执行和状态追踪
- **任务编排**: 复杂任务的工作流编排
- **定时任务**: 基于时间触发的自动化任务

### 🔄 流水线管理
- **流水线配置**: 可视化流水线配置
- **状态机管理**: 流水线状态转换和跟踪
- **执行监控**: 实时监控流水线执行状态

### 🗂️ 仓库集成
- **多平台支持**: GitHub、GitLab、Azure DevOps
- **仓库同步**: 自动同步仓库元数据
- **Git 操作**: 支持克隆、拉取、提交、推送

### 💬 实时聊天
- **AI 对话**: 与 AI Agent 实时对话
- **流式响应**: SSE 流式输出，提升用户体验
- **上下文管理**: 多轮对话上下文保持

### 📚 Wiki 文档系统
- **Markdown 支持**: 原生 Markdown 文档管理
- **版本控制**: 文档版本追踪
- **全文搜索**: 快速检索文档内容

### ⚙️ 系统配置
- **配置管理**: 集中化系统配置
- **权限控制**: 基于角色的访问控制
- **配置加密**: 敏感配置加密存储

### 📊 数据仪表板
- **任务统计**: 任务执行统计和趋势分析
- **Agent 性能**: Agent 执行效率分析
- **系统监控**: 系统资源和健康状态监控

## 技术亮点

- ✅ **前后端分离**: Vue 3 + .NET 10.0，技术栈现代化
- ✅ **模块优先架构**: 前端采用模块优先结构，高内聚低耦合
- ✅ **四层架构**: 后端采用经典四层架构，职责清晰
- ✅ **AI 原生**: 集成 LLM 网关和 Agent 执行引擎
- ✅ **状态机驱动**: 任务和流水线状态管理基于状态机
- ✅ **流式响应**: SSE 流式输出，提升实时性
- ✅ **JWT 认证**: 无状态认证，支持令牌刷新
- ✅ **ORM 集成**: SqlSugar ORM，数据访问简洁高效
- ✅ **开发数据库**: SQLite 开发环境，零配置启动
- ✅ **Swagger 文档**: 自动生成 API 文档，提升开发效率

## 目标用户

- **研发团队**: 希望通过 AI 提升开发效率的团队
- **DevOps 工程师**: 需要自动化流水线和任务编排的工程师
- **项目经理**: 需要实时监控项目进度和任务状态的管理者
- **AI 研究者**: 希望研究 AI Agent 协作和编排的研究人员

## 使用场景

- 🎯 **自动化代码审查**: AI Agent 自动审查代码质量
- 🎯 **智能任务分配**: 基于 Agent 能力自动分配任务
- 🎯 **持续集成/部署**: 自动化 CI/CD 流水线
- 🎯 **知识库管理**: 团队知识沉淀和共享
- 🎯 **项目进度追踪**: 实时监控任务和流水线状态
